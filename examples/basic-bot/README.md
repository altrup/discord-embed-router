# Discord Embed Router Example Bot

A discord bot that showcases how to use `discord-embed-router`

## Setup

### Prerequisites

- Create a Discord bot ([guide](https://discordjs.guide/legacy/preparations/app-setup))

### Usage

- Enter folder

  ```bash
  cd examples/basic-bot
  ```

- Copy [`.env.example`](.env.example) to `.env` in root directory and replace the values

  ```bash
  cp .env.example .env
  ```

- Deploy commands

  ```bash
  npm run deploy-commands
  ```

- Start bot with hot reloading

  ```bash
  npm run dev
  ```

- Invite bot to your server ([guide](https://discordjs.guide/legacy/preparations/adding-your-app))
